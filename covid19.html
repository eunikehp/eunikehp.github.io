Covid-19 Analysis

Project Summary

The purpose of this project was to find out how was the spread of COVID-19 cases around the world. This included the death cases and total vaccinations. The dataset used for this analysis is COVID-19 Deaths data which taken from Our World in Data. It contains information regarding numbers of COVID cases, deaths, and how many people were getting vaccine from Jan 8, 2020 to Apr 30, 2021.

The tools:

  • Microsoft SQL Server Management Studio (SMSS)
  • Tableau

The steps:

  • Source data (CSV from website)
  • Merge dataset using SQL Server
  • Connect to cleaned CSV using Tableau
  • Create dashboard to visualized data
